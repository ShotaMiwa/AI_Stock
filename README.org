* プロジェクト概要
** 目的
- 株価データ＋ニュースデータを用いた
- 1週間先の株価上昇/非上昇予測
- 趣味だが「実運用可能性」を意識した設計

** 基本方針
- End-to-End 学習（株価 + ニュース）
- 時系列整合を最優先
- バックテスト前提での意思決定

* クイック操作 (Emacs)
** 基本の動き (ナビゲーション)
   - [[file:README.org][README]] を開く :: =C-x b= のあと =REA= と打って Enter
   - リンクを開く :: リンク（青字）の上で =C-c C-o=
   - 前の画面に戻る :: =C-c &= (リンクで飛んだ後に戻る時)
   - バッファ(ファイル)切替 :: =C-x b=
   - バッファリスト表示 :: =C-x C-b=

** 閲覧・編集 (Org-mode)
   - 見出しの開閉 :: =TAB=
   - 全見出しの開閉 :: =S-TAB= (Shift + Tab)
   - 保存 :: =C-x C-s=
   - Emacsの終了 :: =C-x C-c=

** 高度な検索 (全ファイル横断)
   - プロジェクト内検索 :: =M-x rgrep=
     - 1. 検索ワードを入力
     - 2. =*.org= と入力 (対象ファイル)
     - 3. Enter (検索場所はデフォルトで現在地)
     - ※ 検索結果上で =Enter= を押すとその場所にジャンプします

** 運用ルール (覚書)
   - 1. =ai_stock= コマンドで全ファイルを開いて作業開始
   - 2. 常に =README.org= を起点にして、リンクから各工程に飛ぶ
   - 3. 終わる時は =C-x C-s= で保存を忘れずに
